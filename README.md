# ✦ Aura Skincare — Proyecto Módulo 4

Proyecto académico individual de la formación en **SoyHenry**.

Consistió en la construcción de un frontend de un  **e-commerce de productos de skincare**, con autenticación de usuarios, carrito de compras y dashboard de pedidos.

🌐 [Ver demo en vivo](https://e-commerce-mod-4.vercel.app/home)

## 🚀 Tecnologías

- **Frontend:** React, Next.js
- **Autenticación:** Auth.js (NextAuth)
- **Deploy:** Vercel
- **Control de versiones:** Git / GitHub

## ⚙️ Funcionalidades principales

- Registro e inicio de sesión de usuarios
- Catálogo de productos de skincare
- Gestión del carrito de compras: agregar, eliminar y visualizar productos seleccionados
- Dashboard de pedidos del usuario
- Deploy en Vercel

## 🛠️ Instalación

1. Clonar el repositorio
```bash
git clone https://github.com/pi-rym/PM4-RocioArias7.git
```
2. Instalar dependencias
```bash
npm install
```
3. Iniciar el proyecto
```bash
npm run dev
```
