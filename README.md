## ⚙️ Funcionalidades Principales

*   **Catálogo Interactivo:** Navegación prolija de productos de skincare con tarjetas detalladas (precios, descripciones e imágenes).
*   **Gestión de Carrito Activo:** Lógica implementada para agregar, eliminar y actualizar cantidades de productos en tiempo real con cálculo automático del total.
*   **Sistema de Autenticación:** Registro e inicio de sesión seguro integrado para usuarios mediante NextAuth.
*   **Dashboard de Pedidos:** Panel privado para el usuario donde puede visualizar el historial de sus compras y el estado de sus pedidos.

---

## 🚀 Tecnologías Utilizadas

*   **Frontend:** React.js, Next.js 
*   **Estilos y Maquetación:** HTML5, CSS3, Tailwind CSS (Diseño adaptativo y Responsive)
*   **Autenticación:** Auth.js / NextAuth
*   **Despliegue / Deploy:** Vercel
